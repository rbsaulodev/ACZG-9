# Linketinder - MVP

Este projeto é um protótipo de sistema de recrutamento que une a praticidade do Tinder com o foco profissional do LinkedIn. Desenvolvido como parte do projeto ZG-Hero.

**Desenvolvedor:** Saulo Brilhante

## Funcionalidades
- [x] Listagem de Candidatos (mínimo 5 pré-cadastrados).
- [x] Listagem de Empresas (mínimo 5 pré-cadastradas).
- [x] Cadastro de novos usuários (Candidatos e Empresas).
- [x] Gestão de competências/skills.

## Tecnologias Utilizadas
- **Linguagem:** Groovy
- **Paradigma:** Orientação a Objetos (Herança, Polimorfismo e Interfaces).

## Como Executar
1. Certifique-se de ter o Groovy instalado em sua máquina.
2. Clone este repositório.
3. No terminal, navegue até a pasta do projeto.
4. Execute o comando:
   ```bash
   groovy Main.groovy
